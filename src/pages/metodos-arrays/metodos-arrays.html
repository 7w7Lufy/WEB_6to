JavaScript Array Methods Cheat Sheet

Anadir y Eliminar (Mutables)

Estos metodos modifican el array original.

Mutante push(): Anade uno o mas elementos al final del array.
[1, 2].push(3); // Resultado: [1, 2, 3]
Mutante pop(): Elimina el ultimo elemento y lo devuelve.
[1, 2, 3].pop(); // Devuelve: 3 | Array: [1, 2]
Mutante unshift(): Anade uno o mas elementos al inicio.
[2, 3].unshift(1); // Resultado: [1, 2, 3]
Mutante shift(): Elimina el primer elemento y lo devuelve.
[1, 2, 3].shift(); // Devuelve: 1 | Array: [2, 3]

Transformacion y Filtrado

map(): Crea un nuevo array aplicando una funcion a cada elemento.
[1, 2].map(x => x * 2); // Resultado: [2, 4]
filter(): Crea un nuevo array con los elementos que cumplen una condicion.
[1, 10, 5].filter(x => x > 4); // Resultado: [10, 5]
slice(): Devuelve una copia de una parte del array (desde inicio a fin, sin incluir fin).
['a', 'b', 'c'].slice(1, 3); // Resultado: ['b', 'c']

Busqueda y Utilidades

find(): Devuelve el primer elemento que cumpla la condicion. Si no hay, devuelve undefined.
[5, 12, 8].find(x => x > 10); // Resultado: 12
includes(): Determina si un array contiene un elemento (devuelve true o false).
[1, 2, 3].includes(2); // Resultado: true
Mutante reverse(): Invierte el orden de los elementos in-place.
[1, 2, 3].reverse(); // Resultado: [3, 2, 1]
